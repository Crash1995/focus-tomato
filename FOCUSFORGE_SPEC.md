# FocusForge — Спецификация продукта

## Что это
Десктопное Electron-приложение для macOS — помодоро-таймер с геймификацией и AI-отчётами. Пользователь запускает таймер, работает 25 минут, записывает что сделал, копит XP/уровни/стрики, в конце дня получает AI-анализ продуктивности и красивую карточку для соцсетей.

---

## Стек
- Electron (последняя стабильная версия)
- HTML + CSS + JS (без фреймворков)
- Node.js fs для хранения данных (JSON-файл на диске)
- OpenRouter API для AI-отчётов
- html2canvas (npm) для экспорта карточки в PNG
- canvas-confetti (npm) для анимации level up
- electron-builder для сборки .dmg

## Настройки Electron (main.js)

- Размер окна: **900x560** (горизонтальное, фиксированное, не ресайзить)
- **Frameless window** с custom titlebar (кнопки закрытия/сворачивания слева, как нативные macOS)
- Фон окна: #0a0a0f
- Заголовок: "FocusForge"
- **Безопасность**: nodeIntegration: false, contextIsolation: true, вся работа с fs и сетью — через preload.js + contextBridge
- Dock badge: когда таймер идёт — показывать оставшееся время через app.dock.setBadge()

## Хранение данных

Файл: `~/Library/Application Support/FocusForge/data.json`

При первом запуске — создать директорию и файл с дефолтной структурой:

```json
{
  "sessions": [],
  "stats": {
    "totalXP": 0,
    "level": 1,
    "currentStreak": 0,
    "bestStreak": 0,
    "lastActiveDate": ""
  },
  "settings": {
    "workDuration": 25,
    "breakDuration": 5,
    "apiKey": ""
  }
}
```

Каждая сессия:
```json
{
  "date": "2025-01-15",
  "time": "10:30",
  "description": "Написал структуру лендинга",
  "xp": 35
}
```

Доступ к данным из renderer — только через preload API:
- `window.api.loadData()` — прочитать data.json
- `window.api.saveData(data)` — записать data.json
- `window.api.sendAIRequest(sessions, apiKey)` — отправить запрос к OpenRouter из main process
- `window.api.saveImage(dataURL)` — открыть dialog.showSaveDialog и сохранить PNG
- `window.api.copyToClipboard(text)` — копировать текст в буфер

## Пользовательский flow

### Шаг 1: Открыл приложение
Видит таймер на 25:00 (не запущен), пустой лог, свои статы. Всё готово к работе — никаких онбордингов.

### Шаг 2: Нажал "Старт"
Таймер пошёл. Круговой прогресс-бар убывает. Dock badge показывает оставшиеся минуты. Кнопка "Старт" меняется на "Пауза".

### Шаг 3: 25 минут прошло → звуковой сигнал → модалка
По центру окна модальное окно:
- Заголовок: "Сессия #N завершена! 🎉"
- Поле ввода: placeholder "Что сделал?" (макс 140 символов)
- Две кнопки: "Готово" (сохраняет с описанием, +35 XP) и "Пропустить" (сохраняет без описания, +25 XP)
- Анимация "+35 XP" всплывает и исчезает

### Шаг 4: Автоматически включается отдых (5 мин)
Таймер переключается на 5:00. Цвет прогресс-бара меняется (зелёный или другой оттенок). Метка: "Отдых".

### Шаг 5: Отдых закончился → звук → таймер готов
Таймер возвращается на 25:00, режим "Работа". Не запускается автоматически — ждёт нажатия "Старт".

### Шаг 6: Повторяешь сессии
Каждая новая сессия добавляется в лог. XP копятся. При достижении порога — level up.

### Шаг 7: Level up
Когда XP достигает порога следующего уровня:
- Конфетти на весь экран (canvas-confetti)
- Текст по центру: "Level Up! Фокусёр 🎉" (2 секунды)
- Уровень и название обновляются в правой колонке

### Шаг 8: Итоги дня
Кнопка "Итоги дня ✨" → если нет API ключа, сначала модалка для ввода ключа → запрос к AI → карточка с отчётом.

### Шаг 9: Следующий день
Лог сессий чистый. XP и уровень сохранены. Стрик пересчитан.

## Геймификация — подробная логика

### XP
- +25 за каждую завершённую рабочую сессию
- +10 бонус если пользователь написал описание (нажал "Готово", а не "Пропустить")
- Итого: 35 XP максимум за сессию, 25 XP минимум

### Уровни
| Уровень | Название     | XP от |
|---------|-------------|-------|
| 1       | Новичок      | 0     |
| 2       | Фокусёр      | 100   |
| 3       | Продуктивщик | 300   |
| 4       | Машина       | 600   |
| 5       | Легенда      | 1000  |

Прогресс-бар показывает заполнение от порога текущего уровня до порога следующего.

### Стрик
- День считается "продуктивным" если завершено **4+ рабочих сессий**
- Логика при запуске приложения:
  1. Прочитать `lastActiveDate` из data.json
  2. Если `lastActiveDate` = сегодня → ничего не делать
  3. Если `lastActiveDate` = вчера → проверить количество сессий за вчера: если 4+ → currentStreak += 1; если < 4 → currentStreak = 0
  4. Если `lastActiveDate` раньше чем вчера → currentStreak = 0
  5. Обновить `bestStreak` если `currentStreak > bestStreak`
  6. Обновить `lastActiveDate` = сегодня
- Отображение: 🔥 N дней (если стрик 0 — показывать 🔥 0 дней, не скрывать)

## AI-отчёт — подробная логика

### Ввод API ключа
При первом нажатии "Итоги дня ✨" (когда apiKey пуст):
- Модалка: "Для AI-отчёта нужен ключ OpenRouter"
- Ссылка: "Получить бесплатно → openrouter.ai/keys" (открывается в системном браузере через shell.openExternal)
- Поле ввода для ключа
- Кнопка "Сохранить" → пишет в data.json

### Запрос (выполняется из main process через IPC)

```
POST https://openrouter.ai/api/v1/chat/completions
Headers:
  Authorization: Bearer [apiKey]
  Content-Type: application/json
Body:
{
  "model": "openai/gpt-4o-mini",
  "messages": [
    {
      "role": "system",
      "content": "Ты — продуктивити-коуч. Получаешь список рабочих сессий пользователя за день (25-минутные блоки помодоро). Напиши краткий энергичный отчёт:\n1) Главное достижение дня (1 предложение)\n2) Паттерн в работе — что заметил (1-2 предложения)\n3) Совет на завтра (1 предложение)\nРусский язык, дружелюбно, энергично, без воды. Максимум 5 предложений."
    },
    {
      "role": "user",
      "content": "Мои сессии за сегодня:\n10:30 — Написал структуру статьи\n11:00 — Сделал 3 варианта заголовка\n11:30 — без описания\n12:00 — Отредактировал финальный текст"
    }
  ]
}
```

### Отображение
Пока ждём ответ — анимация загрузки (пульсирующие точки) в центральной колонке.

Ответ показывается в **карточке**, которая заменяет лог сессий:
- Дата крупно
- Строка: "🍅 N сессий  ⚡ N XP  🔥 N дней"
- Текст AI-отчёта
- Текущий уровень и название
- Лого "FocusForge" мелко внизу
- Кнопка "← Назад к сессиям"
- Кнопка "Сохранить PNG"
- Кнопка "Скопировать текст"

### Обработка ошибок
- Неверный ключ (401) → "Неверный API ключ. Проверь на openrouter.ai/keys"
- Нет интернета → "Нет подключения к интернету"
- Другие ошибки → "Что-то пошло не так. Попробуй ещё раз"

## Экспорт карточки

### PNG (через html2canvas)
- Размер: 1080x1080
- Дизайн: тёмный фон (#0a0a0f), градиентная бирюзовая рамка, крупные цифры
- Содержимое: дата, сессии, XP, уровень, стрик, AI-текст, лого "FocusForge"
- При нажатии "Сохранить PNG" → dialog.showSaveDialog → сохранить файл
- Toast-уведомление: "Сохранено! Поделись #FocusForge"

### Текст (копировать в буфер)
Формат:
```
FocusForge | 15 января 2025
🍅 6 сессий | ⚡ 195 XP | 🔥 3 дня
---
[AI-текст отчёта]
---
#FocusForge
```

## Дизайн

### Цвета
- Фон: #0a0a0f
- Поверхности (карточки, модалки): #1a1a2e
- Бордеры: #2a2a3e
- Основной текст: #e0e0e0
- Второстепенный текст: #888
- Акцент: #00f5d4 (бирюзовый)
- Акцент отдыха: #7bed9f (зелёный)
- XP текст: #f9ca24 (золотой)
- Ошибки: #ff6b6b

### Типографика
- Шрифт таймера: моноширинный (JetBrains Mono или system monospace), крупный ~48px
- Основной текст: system-ui, 14px
- Заголовки: system-ui, 16-18px, bold

### Анимации
- Круговой прогресс: плавное убывание (CSS transition или requestAnimationFrame)
- "+35 XP": fadeIn вверх + fadeOut (CSS keyframes, 1.5 секунды)
- Level up: конфетти (canvas-confetti) + свечение фона (box-shadow pulse)
- Модалки: fadeIn + scale (0.95 → 1.0)
- Hover на карточках сессий: подсветка бордера бирюзовым
- Toast-уведомления: slideIn справа, автоскрытие через 3 секунды

**Задача 4**: Экспорт карточки в PNG (html2canvas) + копирование текста + toast-уведомления

**Задача 5**: Настройки (длительности, ключ) + конфетти при level up + dock badge + иконка + конфиг electron-builder для сборки .dmg
