# FocusForge

FocusForge — macOS-приложение на Electron для фокус-сессий, XP, стриков и AI-итогов дня через OpenRouter.

## Скачать

Скачайте последнюю версию на странице [Releases](https://github.com/Crash1995/focus-tomato/releases).

Файл для macOS называется `FocusForge-0.1.0-arm64.dmg`.

## Установка

1. Откройте скачанный `.dmg`.
2. Перетащите `FocusForge` в папку `Applications`.
3. Запустите `FocusForge` из `Applications`.

Если macOS покажет предупреждение о приложении из интернета, откройте приложение через правый клик → `Open`.

## Что внутри

- Pomodoro-таймер с рабочими сессиями и отдыхом.
- XP, уровни и стрики.
- AI-итоги дня через OpenRouter.
- Копирование текста отчета и экспорт карточки в PNG.
- Локальное хранение данных на вашем Mac.

## OpenRouter

Для AI-итогов нужен OpenRouter API key. Вставьте его в настройках приложения.
